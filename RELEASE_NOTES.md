# Mad Prompters Real Estate Studio — v1.4.0

> *We don't make ads. We make worlds.*

Turn a folder of ordinary listing photos into a **15-second cinematic property video** — generated entirely inside a Claude chat on the Higgsfield engine, in the **style of your choice**, and **morph-free by design** (the house always stays the house).

## What's new in 1.4.0
- 🔥 **Viral cut / FX presets** — **Viral Cut**, **FX Showcase**, **Hype Reel**: fast, beat-synced edits with hard cuts and stylized FX, rendered as **one 15-second clip** (no stitching). Cuts/FX land only on locked frames, so it still can't morph.
- 🎚️ **Creator Mode** — a toggle that adds viral pacing (punch-ins, speed ramps, beat-sync, hard cuts) to any walkthrough style except Story.
- 🔤 **On-screen text, rendered in-clip** — when a cut style or Creator Mode is on, the skill asks what text you want (hook+payoff, stat cards, CTA punches, or custom), confirms the lines, and bakes them into the storyboard so they pop up in the video — no post pass.
- 👆 **Family-first style picker** — 10 presets selected in two quick taps (Walkthrough / Viral-Cut / Recommend).
- 🛡️ **Twilight Glam NSFW fix** — locked to "warm evening / amber / golden-hour" wording so evening renders stop getting flagged and refunded.

*Builds on v1.3.0's Style Preset Library (7 walkthrough looks + style-first intake).*

## Install
1. Download the bundle and unzip it.
2. In Claude: **Settings → Capabilities → Skills → Add Skill**, and upload the `mad-prompters-real-estate-studio` folder (contains `SKILL.md` + `HOW-TO-USE.md`).
3. Enable the **Higgsfield** connector in your chat.
4. Start a new chat and type `/mad-prompters-real-estate-studio` — or just drop your listing photos and ask for a video.

See `mad-prompters-real-estate-studio/HOW-TO-USE.md` for the full guide.
