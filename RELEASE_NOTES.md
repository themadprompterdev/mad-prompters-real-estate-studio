# Mad Prompters Real Estate Studio — v1.5.0

> *We don't make ads. We make worlds.*

Turn a folder of ordinary listing photos into a **15-second cinematic property video** — generated entirely inside a Claude chat on the Higgsfield engine, in the **style of your choice**, and **morph-free by design** (the house always stays the house).

## What's new in 1.5.0
- 🗺️ **Floor Plan Lock (optional)** — attach a floor plan and the skill uses it as a spatial map: it **labels your unlabeled photos by room**, **orders the walkthrough by real adjacencies**, and **tags every transition glide-vs-cut**. It's the strongest anti-morph aid available — and it solves unlabeled photos for you. The room-match confirmation folds into the hero-approval step you already do (no extra back-and-forth), with an opt-in "you-are-here" floor-plan beat (default off).
  - *Honest boundary:* the plan guides how the storyboard is built — the video model can't navigate a blueprint — so the plan image only appears on screen if you opt into the orientation beat. No plan? It falls back to the standard best-guess ordering.

*Builds on v1.4.0 (viral cut/FX presets, Creator Mode, in-clip on-screen text) and v1.3.0 (the 7-look Style Preset Library).*

## Install
1. Download the bundle and unzip it.
2. In Claude: **Settings → Capabilities → Skills → Add Skill**, and upload the `mad-prompters-real-estate-studio` folder (contains `SKILL.md` + `HOW-TO-USE.md`).
3. Enable the **Higgsfield** connector in your chat.
4. Start a new chat and type `/mad-prompters-real-estate-studio` — or just drop your listing photos (and a floor plan, if you have one) and ask for a video.

See `mad-prompters-real-estate-studio/HOW-TO-USE.md` for the full guide.
